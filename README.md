## 🚀 Descripción del Proyecto
El proyecto consiste en un **Moodboard interactivo** que utiliza las capacidades más recientes de CSS para crear un layout dinámico. Se ha priorizado la limpieza del código, la accesibilidad mediante etiquetas semánticas y una experiencia de usuario fluida.

### Características Técnicas:
*   **Maquetación Semántica:** Uso estricto de etiquetas HTML5 (`<main>`, `<section>`, `<article>`, `<figure>`, `<aside>`, `<nav>`, `<footer>`) para garantizar la accesibilidad y validación W3C.
*   **CSS Grid Layout:** Estructura basada en `grid-template-areas` para una gestión eficiente de los bloques en diferentes tamaños de pantalla.
*   **Diseño Responsive:** Adaptación total a dispositivos móviles y tablets mediante Media Queries.
*   **Variables CSS:** Implementación de un sistema de diseño basado en variables para colores, sombras y radios de borde, facilitando la escalabilidad del estilo.
*   **CSS Nesting (Anidamiento):** Uso de la nueva sintaxis de anidamiento nativo para mantener el código organizado y jerárquico.
*   **Interactividad:** Efectos visuales de foco (`hover`) con transformaciones 3D y transiciones suaves en botones y tarjetas.

## 🛠️ Tecnologías Utilizadas
*   HTML5 (Semántico)
*   CSS3 (Variables, Grid, Nesting)
*   Google Fonts (Tipografía 'Outfit')
